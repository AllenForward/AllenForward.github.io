第一篇,如何用Hexo搭建属于自己的博客

发表于 2018-07-09 | 分类于 Hexo

今天,下定决心终于把Hexo搭建好了(虽然过程简单无比),本想着搭好以后随便写点东西记录一下,明天开始写点学习的笔记,忽然想起搭建博客的初衷,觉得自己不应该再懒散下去,遂决定还是写点什么,东西虽然浅显,主要还是希望自己能够坚持下去,加油吧!

准备工作

使用Hexo搭建博客,你需要在你的电脑上做点准备工作:

  1. 安装Git

    • Windows：下载并安装 Git.
    • Mac：使用 Homebrew, MacPorts ：brew install git;或下载 安装程序 安装。
    • Linux (Ubuntu, Debian)：sudo apt-get install git-core
    • Linux (Fedora, Red Hat, CentOS)：sudo yum install git-core

  2. 安装Node.js

    安装 Node.js 的最佳方式是使用 nvm。
    cURL:
    $ curl https://raw.github.com/creationix/nvm/master/install.sh | sh
    wget:
    $ wget -qO- https://raw.github.com/creationix/nvm/master/install.sh | sh
    安装完成后，重启终端并执行下列命令即可安装 Node.js。
    $ nvm install stable
    或者您也可以下载 安装程序 来安装。

  3. 安装Hexo

    所有必备的应用程序安装完成后，即可使用 npm 安装 Hexo。

    npm i -g hexo-cli

  4. 配置SSH

    关于什么是SSH,请自行百度.

    • 本地生成公钥私钥
      $ssh-keygen -t rsa -C “youremail@xx.com“

    • 添加公钥到Github
      cat ~/.ssh/id_rsa.pub
      复制这段你看不懂的密钥

    • 登录 Github，右上角 头像 -> Settings —> SSH keys —> Add SSH key。把公钥粘贴到key中，填好title并点击 Add key。

快速开始

  1. 找到一个你喜欢的文件夹打开Git Bash，输入

    hexo i //init的缩写
    hexo g //generetor的缩写
    hexo s //server的缩写

  2. 打开浏览器输入 localhost:4000查看

    如果在你的浏览器中出现上图的页面，恭喜你，现在你已经可以在这个主题下写博客了。
    但你是不是觉得现在的主题不是你想要的style，行，Hexo有大量的主题可供选择，这里我选择了nexT主题。

更换主题 ——nexT

  1. 在你的站点根目录打开Git Bash，输入

    git clone https://github.com/iissnan/hexo-theme-next themes/next

  2. 克隆完成后，打开站点配置文件找到theme字段，并将值landscape改成next。

  3. 然后Git Bash输入

    hexo clean  //清除缓存
hexo g  //重新生成代码
hexo s  //部署到本地
// 浏览器输入localhost:4000查看

  4. 选择nexT风格
    nexT主题有三种风格选择，应用哪种风格完全取决你。

    • Muse - 默认 Scheme，这是 NexT 最初的版本，黑白主调，大量留白
    • Mist - Muse 的紧凑版本，整洁有序的单栏外观
    • Pisces - 双栏 Scheme，小家碧玉似的清新

    进入主题配置文件，找到scheme字段，将值改成上面三个的一个，你将看到另一个样子的nexT。

    hexo clean
hexo g
hexo s

    如果你有更大的定制需求，你完全可以根据next官方文档进行相应设置。

博客上传到Github

  1. 登录你的github，然后创建一个新的库

    这里的项目名字必须为你的用户名

  2. 创建仓库成功以后点击设置创建一个page

    然后点击choose theme

    点击确定，然后访问链接 https://你的用户名.github.io，这时候你的页面应用的是github选择的主题，接下来我们把Hexo的代码部署到github上。

  3. 修改站点配置文件

  4. 部署
    安装一个插件

    npm install hexo-deployer-git --save

    安装完成后执行

    hexo d

    再次访问 https://你的用户名.github.io

    大功告成了！！！关于如何在hexo新建文章，自己去官网看文档啊，我也是刚入门啊。
